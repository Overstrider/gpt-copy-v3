# codedungeon-test-loop

Use when verification or test repair is the main work.

Loop:
- Identify the smallest failing behavior.
- Run or add the focused failing test.
- Make the smallest fix.
- Re-run the focused test.
- Run the broader relevant suite.
- Record exact commands and outcomes.
